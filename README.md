# Programacion III

Este repositorio contiene todos los ejercicios hechos a lo largo de la cursada de programacion III.

## Lenguajes

Los lenguajes tratados fueron:

* <strong>PHP</strong>
* SQL

## Temas vistos

* Archivos
* Subida de archivos a un servidor
* Base de datos
* PDO
* REST
* SLIM
* Middleware
